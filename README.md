# Ruby::Taxii

Ruby TAXII client

Currently under development - likely to break at a moment's notice.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruby-taxii'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruby-taxii
